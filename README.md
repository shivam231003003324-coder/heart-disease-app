# ❤️ Heart Disease Prediction App

## Overview
A machine learning web application that predicts the likelihood of heart disease using logistic regression.

## Features
- 13 medical input parameters
- 81% prediction accuracy
- Real-time predictions
- User-friendly web interface

## How to Use
1. Enter patient health information using the sliders and dropdowns
2. Click "Predict Heart Disease Risk"
3. Get instant prediction with probability percentage

## Technical Stack
- **Python** - Programming language
- **Scikit-learn** - Machine learning
- **Streamlit** - Web interface
- **Pandas** - Data processing
- **NumPy** - Numerical computing

## Model Performance
- Accuracy: 81%
- Dataset: 1025 patient records
- Features: 13 medical indicators
- Algorithm: Logistic Regression

## Installation (Local)
pip install -r requirements.txt
streamlit run app.py

## Disclaimer
⚠️ This app is for educational purposes only. It should NOT be used for real medical diagnosis. Always consult a healthcare professional.
